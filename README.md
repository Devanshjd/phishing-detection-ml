## Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train models

**URL model** (requires `data/urls.csv` with columns: `url`, `status`):
```bash
python train_model.py
```

**Email model** (requires `data/raw_emails/phishing_email.csv`):
```bash
python train_email_model.py
```

**SMS model** (requires `data/raw_sms/sms_spam.csv`):
```bash
python train_sms_model.py
```

For the SMS dataset, you can use the [UCI SMS Spam Collection](https://archive.ics.uci.edu/dataset/228/sms+spam+collection).

### 3. Run the backend
```bash
uvicorn backend.api:app --reload --port 8000
```

### 4. Open the dashboard
Navigate to `http://127.0.0.1:8000` ‚Äî it will redirect to the cyberpunk frontend.

### Alternative: Streamlit frontend
```bash
streamlit run app.py
```

## API Endpoints

| Method | Endpoint          | Description                    |
|--------|-------------------|--------------------------------|
| POST   | `/predict/url`    | Scan a URL for phishing        |
| POST   | `/predict/email`  | Scan email content             |
| POST   | `/predict/sms`    | Scan SMS message               |
| GET    | `/history/recent` | Get recent scan history        |
| GET    | `/stats`          | Aggregate scan statistics      |
| GET    | `/health`         | Check which models are loaded  |

## Key Improvements (v2)

1. **Feature-based URL model**: Uses 49 hand-crafted features (entropy, TLD, 
   suspicious words, structural signals) fed into RandomForest ‚Äî achieving 93%+ accuracy
2. **Dedicated SMS detection**: Separate model trained on SMS spam data, with 
   automatic fallback to email model if unavailable
3. **Complete FastAPI backend**: All three detection types with proper endpoints,
   input validation, scan history, and statistics
4. **Hybrid scoring**: Allowlist bypass for known-good domains + ML for unknown URLs
5. **Improved requirements.txt**: All dependencies listed with minimum versions

## Models

| Scanner | Algorithm                | Features                        |
|---------|--------------------------|----------------------------------|
| URL     | RandomForest (200 trees) | 49 hand-crafted URL features     |
| Email   | Logistic Regression      | TF-IDF char n-grams (3-5)       |
| SMS     | Logistic Regression      | TF-IDF char n-grams (3-5)       |
